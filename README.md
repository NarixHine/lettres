# Lettres

> Letters of mine, for everyone.

## Tech Stack
- Framework: `Next.js`,
- UI Library: `Chakra UI`,
- Database: `Xata` & `Vercel KV`,
- Authentication: `Clerk`,
- Deployed on `Vercel`,
- Text written in `Markdown`, rendered by [`zenn-markdown-html`](https://github.com/zenn-dev/zenn-editor).
